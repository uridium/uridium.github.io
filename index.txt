--
zwiekszenie ilosci conntrack:
Nie zmieniamy ip_conntrack_max recznie. Zamiast tego trzeba zwiekszyc ip_conntrack_buckets:

modprobe ip_conntrack hashsize=16384

Wtedy ip_conntrack_max powinno automatycznie skoczyc do np 131072.
Jesli nie mozemy wyladowac modulu ip_conntrack, to:

echo 16384 >/sys/module/ip_conntrack/parameters/hashsize
sysctl -w net.ipv4.netfilter.ip_conntrack_max=131072

i sprawdzamy czy dane sa poprawne:

sysctl -a | grep conntrack

-

Obliczmy zuzycie pamieci przez conntrack:

ip_conntrack_max * ilosc_bajtow_dla_jednego_wpisu_w_conntrack + ip_conntrack_buckets * rozmiar_wskaznika

gdzie:
ilosc_bajtow_dla_jednego_wpisu_w_conntrack to:
 awk '/ip_conntrack /{print $4}' /proc/slabinfo
rozmiar_wskaznika to 8 bajtow na i386

In practice, pointers will be size 2 on a 16-bit system (if you can
find one), 4 on a 32-bit system, and 8 on a 64-bit system, but there's
nothing to be gained in relying on a given.

te informacje mozna tez zobaczyc w /var/log/kern.log:
ip_conntrack version 2.4 (16384 buckets, 131072 max) - 224 bytes per conntrack

-

Obliczmy:
ile moze wynosic ip_conntrack_max przy danej ilosci RAMu:

ip_conntrack_max = RAMSIZE (in bytes) / 16384 / (x / 32)
where x is the number of bits in a pointer (for example, 32 or 64 bits)

ile moze wynosic ip_conntrack_buckets przy danej ilosci RAMu:

ip_conntrack_buckets = ip_conntrack_max / 8 = RAMSIZE (in bytes) / 131072 / (x / 32)
where x is the number of bits in a pointer (for example, 32 or 64 bits)

No i:
ip_conntrack_max = ip_conntrack_buckets * 8

ext3grep, odzyskiwanie plikow:

1. odzysk pliku 'dwa'

- odmontowujemy partycje (tu: /dev/vg/bla)
- ext3grep /dev/vg/bla --dump-names
- ext3grep /dev/vg/bla --restore-file dwa
 (uwaga, jesli 'dwa' to byl katalog z zawartoscia, to odzyskany zostanie sam katalog)
- plik odzyskany zostanie do bierzacego katalogu, do RESTORED_FILES

mozemy tez sprawdzic ktore pliki zostaly skasowane:
- ext3grep /dev/vg/bla --search dwa
- ext3grep /dev/vg/bla --ls --block 579
jesli teraz widzimy D, tzn ze Deleted

2. odzysk wszystkich plikow

- ext3grep /dev/vg/bla --restore-all

rpm cheatsheet

rpm -qa                             - List of installed packages
rpm -e <package>                    - Uninstall package from system
rpm -qf /path/to/file               - Which package put this file here

rpm -q --whatprovides <package>     - List of packages provided by the package
rpm -q --whatrequires <package>     - List of packages required by the package
rpm -q --provides <package>         - List of files provided by the package
rpm -q --requires <package>         - List of files required by the package
rpm -qp --provides package.rpm      - List of files provided by the non-installed package
rpm -qp --requires package.rpm      - List of files required by the non-installed package

rpm -ql <package>                   - List of files in an installed package
rpm -qlp package.rpm                - List of files in a non-installed package

rpm -q <package>                    - Returns full name and version of installed package
rpm -qp package.rpm                 - Returns full name and version of non-installed package

rpm -qi <package>                   - Returns misc details about package
rpm -qip package.rpm                - Returns misc details about non-installed package
